# pipeline topics get

## Overview

Get a local topic

**Aliases:** `get`

**Usage:**

```
quix pipeline topics get [<topic-name>] [options]
```

**Arguments:**

- `<topic-name>` : The name of topic

**Options:**

- `-o, --output <json|table>` : Output type of the retrieved data
- `--directory <directory>` : Base directory (defaults to current directory)
